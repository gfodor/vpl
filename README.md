# Vampire Free License (VFL)

The Vampire Free License is a source-available software license based on the
Apache License, Version 2.0, with additional restrictions.

## What it does

The VFL grants the same rights as the Apache License 2.0, except that those
rights are withheld from a defined set of **Restricted Organizations** and any
current employee of those organizations (a **Restricted Person**).

As of Version 1.0, the Restricted Organizations are:

- Meta Platforms, Inc. (including Facebook)
- Oracle Corporation
- Anthropic, PBC

Restricted Persons may not use, copy, modify, distribute, or otherwise exploit
software released under this license, and rights terminate automatically if a
recipient later becomes a Restricted Person.

## Purpose

The license exists to let software be released to the open source community
while reducing the risk of it being incorporated by organizations that have
been hostile to open source and/or independent developers.
